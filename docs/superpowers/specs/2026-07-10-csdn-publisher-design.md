# CSDN 草稿发布 Skill 设计

## 目标

创建个人 Skill `csdn-publisher`，读取本地 Markdown 文章，复用用户当前 Chrome 中的 CSDN 登录状态，将文章保存到 CSDN 草稿箱。

首版只负责创建草稿，不公开发布、不定时发布、不覆盖已有草稿，也不处理登录凭据。

## 使用方式

用户提供一个本地 Markdown 文件路径，例如：

```text
使用 $csdn-publisher 将 /absolute/path/article.md 保存到 CSDN 草稿箱
```

输入文件必须可读，并包含至少一个一级标题。Skill 取第一个一级标题作为 CSDN 文章标题，其余内容作为正文。

## 方案选择

采用控制用户当前 Chrome 的方式，复用已有 CSDN 登录会话。该方式无需在 Skill 中存储密码、Cookie 或 Token，也符合“使用前由用户自行登录”的要求。

不采用以下方案：

- Playwright 独立浏览器：需要维护单独的浏览器登录状态。
- 只生成可粘贴内容：不能完成自动保存草稿的目标。

## 处理流程

1. 接收 Markdown 文件的绝对路径。
2. 校验文件存在、可读且可按 UTF-8 解码。
3. 提取第一个一级标题作为文章标题。
4. 处理正文中的图片：
   - 保留 HTTP 或 HTTPS 网络图片语法。
   - 将本地图片替换为原位置占位符 `【待上传图片：原始路径】`。
5. 控制当前 Chrome 打开 CSDN 创作中心。
6. 检查 CSDN 登录状态；未登录时立即停止并提示用户手动登录。
7. 检查草稿箱是否已有同名标题；存在时停止，不覆盖也不新建重复草稿。
8. 新建 Markdown 文章并填写标题和处理后的正文。
9. 执行保存草稿。
10. 根据明确的成功提示或草稿箱记录验证保存结果。

只有验证成功后才能报告草稿已保存。

## 组件

```text
mine/csdn-publisher/
├── SKILL.md
├── agents/
│   └── openai.yaml
└── scripts/
    └── prepare_markdown.py
```

### SKILL.md

定义触发条件、输入要求、Markdown 预处理命令、Chrome 操作流程、保存验证方式和强制停止条件。明确禁止点击发布、定时发布或覆盖已有草稿。

### scripts/prepare_markdown.py

提供确定性的本地预处理：

- 接收 Markdown 文件路径。
- 提取标题。
- 保留代码块和普通 Markdown 内容。
- 保留网络图片。
- 将本地图片替换为占位符。
- 以结构化格式输出标题和正文，供后续浏览器操作使用。

脚本不访问网络，不读取浏览器数据。

### agents/openai.yaml

提供 Skill 的展示名称、简短说明和包含 `$csdn-publisher` 的默认调用示例。

## 安全边界

- 不读取、记录或保存密码、Cookie、Token。
- 不自动填写登录信息，不绕过验证码或其他身份验证步骤。
- 未登录时停止，等待用户自行登录后重新调用。
- 不覆盖同名草稿，不自动生成重复草稿。
- 不点击公开发布、定时发布及其他会公开文章的操作。
- 页面结构、按钮含义或保存结果不明确时立即停止。

## 错误处理

- 文件不存在、不可读或编码异常：报告具体错误并停止。
- 找不到一级标题：要求用户为文章添加一级标题并停止。
- Chrome 不可用：报告浏览器不可操作并停止。
- CSDN 未登录：提示用户在 Chrome 手动登录并停止。
- 发现同名草稿：报告标题并停止。
- 页面结构无法识别：报告当前页面状态并停止，不尝试猜测点击。
- 保存结果无法确认：报告“保存状态未知”，不得宣称成功，也不得再次保存。

## 验证

### 本地验证

- 校验一级标题提取。
- 校验正文和代码块保持不变。
- 校验 HTTP/HTTPS 网络图片保持不变。
- 校验相对路径、绝对路径和 `file://` 本地图片被替换为占位符。
- 校验文件不存在、非 UTF-8 文件和缺少一级标题时返回非零状态。
- 使用 Skill 校验工具检查目录名、YAML frontmatter 和元数据。

### Chrome 实测

Chrome 实测前通知用户，并要求用户确认已在当前 Chrome 登录 CSDN。实测只创建一篇草稿，验证草稿箱存在对应标题和内容；不得执行公开发布。

## 首版范围外

- 自动登录或管理登录凭据。
- 上传本地图片。
- 自动填写标签、分类、摘要或封面。
- 编辑、覆盖或删除已有草稿。
- 批量、定时或公开发布文章。
- 登录失效后的自动重试。
